# 🎓 Student Management System (Python)

## 📌 Project Overview
The **Student Management System** is a console-based Python application used to manage student records efficiently.  
It performs basic **CRUD operations (Create, Read, Update, Delete)** using Python data structures and follows a **menu-driven interface**.

This project is beginner-friendly and suitable for **internship, academic, and portfolio use**.

---

## 🚀 Features
- Add student details (ID, Name, Class, Age)
- Display all student records
- Search student details using ID
- Update existing student information
- Delete student records
- Continuous menu-driven operation

---

## 🛠️ Technologies Used
- **Language:** Python  
- **Data Structures:** List, Dictionary  

### Concepts
- Functions  
- Loops  
- Conditional Statements  
- User Input Handling
